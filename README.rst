June
=======

June is a forum, and this version is based on Flask.


Get involved
--------------

1. fork this repo
2. clone your repo
3. develop on flask branch
4. send your pull request

::

    $ git clone git@github.com:/your-name:june
    $ cd june
    $ git checkout flask


Get started
-------------

::

    $ make database
    $ make babel-compile
    $ make server
